World Wide Web

World Wide Web é um sistema de documetos dispostos na internet que permitem o acesso às informações apresentadas no formato de hipertexto, para ter acesso a tais informações é necessário utilizar um programa de computador chamado de navegador. Os navegadores mais utilizados são: Google Chrome, Mozilla Firefox, Safari, Internet Explorer, Opera, entre outros. Hipertextos são textos exibidos em formato digital, os quais podem conter informações em formato de imagens, sons, vídeos, etc. O acesso a tais informações se dá por meio de links, que servem como uma ponte entre os mais diversos sites da internet e seus conteúdos. A web é composta por documentos HTML - ou Hypertext Markup Language- que é a linguagem que os computadores usam para interpretar páginas web.

Como funciona a web

O cliente (indivíduo, computador e o navegador) faz solicitações à internet para os servidores (Computadores que hospedam os arquivos que compõem a web), o servidor encontra a página solicitada e envia a mesma para o navegador, o navegador resgata a página em formato HTML e, por fim, compacta em formato visual para o usuário. O protocolo HTTP é utilizado pelo navegador para se comunicar com os servidores. O principal tipo de documento da web é o HTML.

HTML

HTML é um editor de hipertexto, utilizado para criação de páginas online e aplicações web.
HTML é, basicamente, o coração da web, ou seja, é a linguagem base da internet que mantém todo o conteúdo da mesma juntos (textos simples, imagens, vídeos e músicas).
HTML é formada por: conteúdo de texto; marcação, que é a aparência desse conteúdo ou como ele é organizado; referências a outros conteúdos, como imagens e vídeos; e links para outras páginas como, por exemplo: https://www.youtube.com

Tags < > < / >

HTML é uma linguagem baseada em marcação; você marca o elemento para mostrar quais informações será exibida na página. Os elementos em HTML consistem em uma tag de abertura (< >) e uma tag de fechamento (< / >) com algum conteúdo entre elas como, por exemplo, < tag de abertura > Oi, como vai você? < /tag de fechamento>

Exemplo de tag:< b >Oi, como vai você?< /b >

Podemos observar que o nome de ambas as tags são os mesmos, entretanto, vale ressaltar que a tag de de fechamento contém uma barra < / >, ao contrario da tag de abertura < > que não contém barra.
Podemos nos referir ao conjuto de < > e < / > como elemento(s).

Observações

Vale ressaltar que é necessário sempre fechar as "tags com conteúdo" (< / >), enquanto as tags vazias como < br > e < img... > não necessita de fechamento.

Diferença entre tag vazia e tag com "conteúdo"

Tag vaizia é uma tag que não tem conteúdo, por exemplo, a tag < br > não tem nenhum tipo de conteúdo, pois sua função é de "quebrar" a linha - lembrando que uma tag vazia não tem fechamento < / > -; enquanto a tag de "conteúdo" são aquelas que, obviamente, tem algum tipo de conteúdo em seu meio, por exemplo < em >olá< /em >.

Inline e Block

Inline - elementos inline é o que está no texto, por exemplo: bom dia < br (elemento inline -quebra de linha) > você não respondeu o meu e-mail.
block - são os elementos que criam uma "caixa" ao redor do texto, por exemplo:

< p (elemento block - paragrafo) >
"Um produto é algo feito em fábricas; uma marca é algo que é comprado pelo consumidor. Um produto pode ser copiado por um concorrente; uma marca é única. Um produto pode ficar rapidamente obsoleto; uma marca de sucesso é eterna".
< /p >
